# 🛡️ PyShield Firewall

A simple Personal Firewall developed using **Python** and **Linux iptables**. This project demonstrates basic firewall management by allowing users to block and unblock IP addresses through a menu-driven command-line interface.

---

## 📌 Features

* ✅ Block incoming IP addresses
* ✅ Unblock previously blocked IP addresses
* ✅ View current firewall rules
* ✅ View firewall activity logs
* ✅ Flush all firewall rules
* ✅ Validate IP addresses before applying rules
* ✅ Log all firewall actions with timestamps
* ✅ Simple and user-friendly command-line interface

---

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/123DarshanM/pyshield-firewall.git
```

Go to the project directory:

```bash
cd pyshield-firewall
```

Run the application:

```bash
sudo python3 firewall.py
```
